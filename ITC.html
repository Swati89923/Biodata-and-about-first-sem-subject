HTML, CSS, and JavaScript

HTML

Learned about different tags in HTML:

  1. html - Root element
  2. head - Document metadata
  3. body - Document content
  4. p - Paragraph
  5. b - Bold
  6. u - Underline
  7. i - Italic
  8. Heading tags (h1 to h6)
  9. ul - Unordered list, ol - Ordered list, li - List item
  10. a - Hyperlink
  11. img - Image
  12. br - Line break
  13. div - Division

CSS

CSS stands for Cascading Style Sheets. It's a styling language used to control the layout and appearance of web pages written in HTML.

Basic CSS Concepts
  1. Selectors
  2. Properties (e.g., color, font-size)
  3. Values (e.g., red, 18px)
  4. Rules (e.g., h1 { color: blue; })
  5. Cascading (inline, internal, external)

JavaScript

Concepts we have studied:

  2. Data types (number, string, boolean, array, object)
  3. Functions (function declarations, function expressions)
  4. Events (click, hover, submit)
